# GoogleCalNotification

A native macOS desktop application for Google Calendar notifications.

## Project Overview

This application is built using SwiftUI to provide seamless calendar notifications on macOS.

## Tech Stack

- **Platform:** macOS
- **Framework:** SwiftUI
- **Language:** Swift

## Setup Instructions

1. Open the project in Xcode.
2. Ensure you have the necessary permissions for Calendar access.
3. Build and run the project.
